# search-music-tool
A new tool to search music in your computer

## How to install
1. **Windows:** run `*.exe` file directly
2. **Linux:** execute `*.py` file
    - Used package: `tkinter`, `mutagen`, `PIL`.
    - Use Python 3 to make the code.
